## How to contribute

### env

- nodejs > 12 (**important**)
- `npm install -g electron-builder` if you need bundle application

### dev

```shell
git clone https://github.com/xcodebuild/iproxy
cd iproxy
yarn run install-deps
yarn run dev
```
